Object-oriented wrapper framework for SDL3, written in C/C++ and designed for quick prototyping and deployment of interactive audio-visual applications.

# NOTES:
## Required Libraries:
- SDL3 (https://github.com/libsdl-org/SDL/releases)
- SDL3_image (https://github.com/libsdl-org/SDL_image/releases)
- SDL3_ttf (https://github.com/libsdl-org/SDL_ttf/releases/preview-3.1.0)
